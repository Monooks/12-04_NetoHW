### Задание 5*

Найдите фильмы, которые ни разу не брали в аренду.

#### ОТВЕТ:
```sql
SELECT f.title
FROM film f
LEFT JOIN inventory i ON i.film_id = f.film_id
LEFT JOIN rental r ON r.inventory_id = i.inventory_id
WHERE r.rental_id is NULL;
```
